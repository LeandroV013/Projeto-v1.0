Lista de Compras — v0.1

Projeto simples de lista de compras em Python, desenvolvido com o objetivo principal de praticar lógica de programação, estruturas básicas da linguagem e versionamento com Git/GitHub.
Este projeto está em evolução contínua e faz parte do meu processo de aprendizado prático, simulando uma rotina real de desenvolvimento, mesmo sendo um projeto individual.

Funcionalidades atuais
    Inserir itens na lista de compras
    Permite inserir um ou vários itens de uma vez, separados por vírgula
    Listar todos os itens adicionados
    Remover itens específicos da lista
    Interface via terminal (CLI)

Tecnologias utilizadas
    Python
    Git e GitHub (controle de versão)

Objetivo do projeto
    Evoluir em Python
    Criar o hábito de versionar código com commits frequentes
    Evoluir o projeto gradualmente, adicionando novas funcionalidades conforme o aprendizado

Próximos passos (planejados)
    Melhorar validações de entrada
    Persistência de dados (JSON ou banco de dados)
    Refatoração do código
    Possível criação de interface gráfica ou web (HTML/CSS ou framework)
